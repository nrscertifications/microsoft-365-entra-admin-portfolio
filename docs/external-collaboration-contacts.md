# External Collaboration & Contacts

This work separates two common external-collaboration objects: a Microsoft 365 contact and a Microsoft Entra B2B guest user.

## Work Completed

- Created and confirmed an external contact in the Microsoft 365 admin center.
- Configured a Microsoft Entra B2B guest invitation with profile, message, and redirect settings.
- Reviewed the guest configuration before invitation.
